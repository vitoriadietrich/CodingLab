# Product Card Design using HTML & CSS | CodingLab

Um projeto simples de **card de produto** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **product card moderno**, ideal para e-commerce e vitrines de produtos.

- HTML
- CSS

## 🎯 Funcionalidades

- Card de produto estilizado
- Design moderno e limpo
- Layout responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
